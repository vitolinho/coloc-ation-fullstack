# Projet coloc'ation
Projet d'école, HETIC Web 2.
# Description
Application de gestion de frais pour des collocations réaliser avec Docker, Php et React.

# Docker
dans le dossier /back,
```
docker compose -d --build
```

# React
dans le dossier /front,
```
yarn start
```
